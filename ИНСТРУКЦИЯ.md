# Инструкция по запуску проекта

## Шаг 1: Установка зависимостей

Откройте терминал в директории проекта и выполните:

```bash
npm install
```

Это установит все необходимые пакеты:
- Next.js 14
- React 18
- TypeScript
- Framer Motion
- Zustand
- TailwindCSS

## Шаг 2: Запуск development сервера

После установки зависимостей запустите:

```bash
npm run dev
```

## Шаг 3: Открыть в браузере

После запуска откройте в браузере:
```
http://localhost:3000
```

## Дополнительные команды

- `npm run build` - собрать production версию
- `npm run start` - запустить production сервер
- `npm run lint` - проверить код линтером

## Если возникли проблемы

1. Убедитесь, что Node.js установлен (версия 18 или выше)
2. Проверьте, что вы находитесь в правильной директории проекта
3. Удалите `node_modules` и `package-lock.json`, затем выполните `npm install` заново
